# BetterFeeds: A Farm Feed Optimizer

**BetterFeeds** is a modern web platform designed to improve how users consume and manage their feeds. It focuses on a cleaner experience, better content organization, and faster access to what matters most.

🌐 Live Site: https://better-feeds.vercel.app

---

## ✨ Features

- 📡 **Streamlined Feed Experience** – Enjoy a cleaner, distraction-free layout  
- ⚡ **Fast Performance** – Optimized for speed and smooth navigation  
- 🎯 **Focused Content** – Prioritizes relevant and meaningful updates  
- 📱 **Responsive Design** – Works seamlessly on desktop and mobile  
- 🧠 **Better Organization** – Helps users manage and view content more efficiently  

---

## 🛠️ Tech Stack

BetterFeeds is built using modern web technologies:

- **Frontend:** React (TypeScript)
- **Styling:** Tailwind CSS  
- **Deployment:** Vercel  

---

## 📚 Mathematical Foundation

Uses Linear Programming to minimize cost and meet the nutritional requirements needed.

**Subject to Constraints:**
1. Nutrient Requirements
2. Ingredient bounds (min/max inclusions)
3. Total feed weight constraint
